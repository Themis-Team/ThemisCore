Detailed Description

The uncertainty is \( \sigma = \sqrt{ \Re(\sigma_V)^2 + t + f |V| } + i \sqrt{ \Im(\sigma_V)^2 + t + f |V| } \), corresponding to the "loose change" model.

Parameter list:

  • parameters[0] ... Threshold to add in Jy.
  • parameters[1] ... Fraction of visibility amplitude to add.
