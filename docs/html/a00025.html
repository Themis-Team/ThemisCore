Detailed Description

Provides basic functionality for a command-line option object, supporting multiple flags, required/optional arguments, and online help information. Because it contains purely virtual functions, this class cannot be instantiated. However, child classes that implement specific types, inherit the relevant functionality and interfaces.

Warning
This class contains multiple purely virtual functions, making it impossible to generate an explicit instantiation.
