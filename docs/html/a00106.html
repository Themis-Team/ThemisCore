Detailed Description

The image is computed by the external radiative transfer code raptor.

Parameter list:
parameters[0] ... Density normalization/scale inside raptor.
parameters[1] ... Inclination angle inside raptor in radians.
parameters[2] ... Isothermal electron temperature assumed in the jet/funnel region where ideal MHD is not to be trusted.
parameters[3] ... Azimuthal viewing angle inside raptor.

Generates and returns rectalinear grid of intensities from an image model generated by the external radiative transfer code raptor in Jy/pixel located at pixels centered on angular positions alpha and beta, both specified in radians and aligned with a fiducial direction. Note that the parameter vector has had the position removed.
