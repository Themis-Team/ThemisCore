Detailed Description

The uncertainty is \( \sigma_{RR,LL,RL,LR} = \sqrt{ \Re(\sigma_V)^2 + t^2 + f^2 |V|^2 + a^2/[1+(|u|/u_0)^b] } + i \sqrt{ \Im(\sigma_V)^2 + t^2 + f^2 |V| + a^2/[1+(|u|/u_0)^b] } \), corresponding to the "broken_power change" model.

Parameter list:

  • parameters[0] ... Threshold to add in Jy, \(t\).
  • parameters[1] ... Fraction of Stokes I visibility amplitude to add, \(f\).
  • parameters[2] ... Noise value in Jy, \(a\) either at zero baseline or at 4 Glambda.
  • parameters[3] ... Baseline length of break in G \( lambda \), \(u_0\).
  • parameters[4] ... Long-baseline broken_power law index, \(b\).
