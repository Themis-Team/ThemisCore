Detailed Description

Given a list of image file names appropriate for the model_image_score class, the variance of the visibility amplitudes are computed at a set of radial and radial locations. These are then added in quadrature to the existing error budget for a given data object. Note that these errors are static, i.e., they are set at construction time and do not vary subsequently.

Warning
Only visibility amplitude and closure phase are implemented. Others are trivial but not yet done.
