Detailed Description

author Paul Tiede

Date
March, 2018

Creates an snapshot of a shearing spot using the VRT2 semi-analytic shearing spot class with the Johannsen 2013 metric. Model uses a two-parameter vector field described below to model an infalling accretion flow. The spots initial densoty profile is taken to be a Gaussian spot with profile

\[ n_0\exp(-\Delta r^{\mu}\Delta r_{\mu}/2R_s - (u^{\mu}r_{mu})^2/2R_s)\]

. Currently the model uses the sed from the 2010 obsevation of Sgr A* for the spot's ectron population.
